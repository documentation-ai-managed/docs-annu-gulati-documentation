## Migration and Import

Move existing documentation to Annu Gulati Documentation with our migration tools.

<Tabs>
  <Tab title="From Confluence" icon="file-text">
    Import pages from Confluence with formatting preserved.
  </Tab>
  <Tab title="From GitHub Wiki" icon="github">
    Pull documentation from GitHub wikis automatically.
  </Tab>
  <Tab title="Custom Import" icon="upload">
    Use our API for custom migration scripts.
  </Tab>
</Tabs>

<CodeGroup tabs="Shell,Python">
  ```bash
  # Bulk import from directory
  annu-docs import --source ./docs --format markdown
  ```
  ```python
  import annu_gulati_docs as docs

  # Custom migration
  client = docs.Client()
  client.import_documents(
      source_path='./legacy-docs',
      format='markdown',
      preserve_structure=True
  )
  ```
</CodeGroup>

<Expandable title="Migration Checklist" default-open="false">
  - Backup existing documentation
  - Test import process with sample data
  - Verify formatting and links
  - Update internal references
  - Train team on new platform
</Expandable>
